# ADEMJOBConnect

Secure ETL pipeline for Luxembourg’s ADEM data, built with R, GitHub Actions, and CI/CD best practices.

## Features
- 🔐 GPG-signed commits
- ⚙️ Jenkinsfile for CI
- 📦 renv for reproducible R environments
- 🧪 SQL schema for structured data

## Getting Started
1. Clone the repo
2. Run `renv::restore()`
3. Execute `etl_pipeline.R`
